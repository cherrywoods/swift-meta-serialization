CodingStackStatus

public enum CodingStackStatus

The possible statuses of a CodingStack.

  • pathMissesMeta

    First status of a new CodingStack. Expresses that a new meta needs to be added or a codingKey needs to be removed to proceed.

    Swift

  • pathFilled

    If a CodingStack has this status, it currently contains at least one meta and waits for a new coding key, so another meta may be added.

    Expresses, that a coding key may be added, or a meta removed. This status expresses some validity of the stack, respectively that it is not waiting for a meta to be pushed.
